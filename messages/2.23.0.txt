SystemVerilog 2.23.0 Changelog:


## Enhancement ###
  - Navigation: navigation bar to display class members (Contribution from Dillan Mills)
    * Display base class which can be expanded to the full inheritance tree by clicking on the +
    * Display methods arguments (folded by default, use arrow in the gutter to unfold)
    * Double cliking on a method name jump to the definition
    * Double cliking on a class name (from base class or members type) immediately open the file
    * Auto-updated when changing file (file type configurable with setting sv.navbar_update)

## Bug Fix ##
  - Parsing:
    * Fix potential deadlock created by extern static function


## Keybind Update ##
To use the new function to display the class hierarchy:

```
    {
        "keys": ["f1"], "command": "verilog_show_navbar",
        "context":
        [
            { "key": "selector", "operator": "equal", "operand": "source.systemverilog"}
        ]
    },
    {"keys": ["ctrl+f1"], "command": "verilog_close_navbar"}
```
